# shadowPanel

`shadowPanel` モジュールは、シャドウオーバーレイパネルを制御します -- 雰囲気演出やフォーカス暗転に使用される、フルスクリーンの透明レイヤーです。

## インポート

```typescript
import { shadowPanel } from "@hmcs/sdk";
```

## 関数

| 関数 | 説明 |
|----------|-------------|
| [alpha](./alpha) | シャドウパネルの現在の透明度を取得します |
| [setAlpha](./setAlpha) | シャドウパネルの透明度を設定します |

参照: [型定義](./types)
