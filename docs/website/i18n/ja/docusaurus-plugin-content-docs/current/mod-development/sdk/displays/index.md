# displays

接続されたモニターの情報を照会します -- 識別子、名前、スクリーン空間のフレーム矩形。

## インポート

```typescript
import { displays } from "@hmcs/sdk";
```

## Functions

| Function | 説明 |
|----------|-------------|
| [findAll](./findAll) | 接続されたモニターごとに 1 つの `GlobalDisplay` オブジェクトの配列を返す |

See also: [Coordinates 型定義](../coordinates/types#globaldisplay)
