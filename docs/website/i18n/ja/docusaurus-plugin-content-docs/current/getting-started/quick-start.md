# クイックスタート

:::tip[始める前に]
[インストール](/getting-started/installation)ガイドを完了していることを確認してください。Desktop Homunculus と公式 MOD がインストールされ、アプリが起動している必要があります。
:::

## Elmerの表示

公式 MOD がインストールされた状態で Desktop Homunculus を起動すると、デスクトップに **Elmer** が表示されます。Elmer はデフォルトのキャラクターで、他のアプリケーションの上に浮かぶ透明ウィンドウ内にレンダリングされる VRM 3D モデルです。

Elmer には `@hmcs/elmer` と `@hmcs/assets` MOD によるアニメーションと動作が組み込まれています。キャラクターはアイドル状態になり、掴まれると反応し、カーソルに追従します。

## キャラクターとのインタラクション

### ドラッグ＆移動

キャラクターをクリックしてドラッグすると、デスクトップ上の好きな位置に移動できます。ドラッグ中、Elmer は「drag」アニメーションに切り替わります。離すと新しい位置にキャラクターが配置されます。

### キャラクターの状態

キャラクターはさまざまな状態に応じて、それぞれのアニメーションで反応します：

| 状態        | トリガー                               |
| ----------- | -------------------------------------- |
| **idle**    | デフォルト状態                         |
| **drag**    | キャラクターをクリック＆ドラッグ       |
| **sitting** | ウィンドウの端にキャラクターをドロップ |

### コンテキストメニュー

キャラクターを右クリックすると、コンテキストメニューオーバーレイが開きます。ここから設定や、インストール済み MOD が提供するその他のアクションにアクセスできます。

### 設定

右クリックのコンテキストメニューから設定パネルを開きます。設定 UI では、アプリケーションや MOD 固有のオプションを構成できます。

### アプリの終了

1. **システムトレイ**（通知領域）にある **Desktop Homunculus** のアイコンを探す
2. トレイアイコンをクリックしてトレイメニューを開く
3. **Exit** を選択

:::tip
`@hmcs/app-exit` MOD がシステムトレイの Exit オプションを提供します。特に Windows ではアプリがタスクバーや Alt+Tab に表示されないため、この MOD がインストールされていることを確認してください。
:::

## 公式 MOD の紹介

Desktop Homunculus には `@hmcs` スコープ配下の公式 MOD セットが提供されています：

| MOD                        | 説明                                                                                           |
| -------------------------- | ---------------------------------------------------------------------------------------------- |
| `@hmcs/elmer`              | デフォルトキャラクター（Elmer）を制御 — キャラクターの生成、アニメーションとカーソル追従の管理 |
| `@hmcs/assets`             | デフォルトの VRMA アニメーション（`idle-maid`、`grabbed`、`idle-sitting`）と効果音             |
| `@hmcs/character-settings` | コンテキストメニューからアクセスできるキャラクターごとの設定パネル                             |
| `@hmcs/settings`           | システムトレイからアクセスできるアプリケーション設定パネル                                     |
| `@hmcs/app-exit`           | システムトレイの終了メニュー（Windows では必須）                                               |
| `@hmcs/menu`               | 右クリックのコンテキストメニューオーバーレイ                                                   |

## 追加 MOD

オプションの MOD で Desktop Homunculus を拡張できます。CLI を使っていつでもインストールできます：

| MOD              | 説明                                                                      |
| ---------------- | ------------------------------------------------------------------------- |
| `@hmcs/voicevox` | [VoiceVox](https://voicevox.hiroshiba.jp/) エンジンを使用した音声合成連携 |

:::info[VoiceVox のセットアップ]
`@hmcs/voicevox` MOD を使用するには、VoiceVox エンジンを別途インストールして起動しておく必要があります。インストール方法は [VoiceVox の Web サイト](https://voicevox.hiroshiba.jp/)をご覧ください。
:::

```shell
hmcs mod install @hmcs/voicevox
```
