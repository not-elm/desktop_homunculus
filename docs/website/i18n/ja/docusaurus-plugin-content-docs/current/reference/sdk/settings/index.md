# settings

`settings` モジュールは、アプリケーション全体のレンダリング設定の読み取りと更新を行います。

## インポート

```typescript
import { settings } from "@hmcs/sdk";
```

## 関数

| 関数 | 説明 |
|----------|-------------|
| [fps](./fps) | 現在のレンダリングフレームレートを返します |
| [setFps](./setFps) | レンダリングフレームレートを更新します |

関連: [型定義](./types)
