# @hmcs/persona

ペルソナ MOD（`@hmcs/persona`）は、ペルソナのライフサイクル管理を行います。起動時のペルソナ生成、アニメーション制御、設定・管理 UI を提供します。

## 概要

Desktop Homunculus が起動すると、ペルソナ MOD のサービスが自動的にメタデータで `auto-spawn: true` が設定されたすべてのペルソナを生成し、アイドルアニメーション、カーソル追従、インタラクション動作を設定します。

## 機能

### サービス（自動起動）

1. **起動時にペルソナを生成** — メタデータの `auto-spawn: true` が設定されたペルソナをすべて生成
2. **アイドルアニメーション**（`vrma:idle-maid`）を VRM 付きペルソナごとにループ再生
3. **カーソル追従** — 各ペルソナの目がマウスの位置を追跡
4. **インタラクションへの応答：**
   - **ドラッグ** — 掴まれたポーズ（`vrma:grabbed`）に切り替わり、カーソル追従を停止
   - **ウィンドウの端に座る** — 座りアニメーション（`vrma:idle-sitting`）に切り替わる
   - **離す** — アイドルに戻り、カーソル追従を再開

### ペルソナごとの設定パネル

以下の方法で設定パネルを開きます：

- **キャラクターを右クリック**して、[コンテキストメニュー](./menu)から **「Persona」** を選択
- パネルはキャラクターの横にフローティング WebView ウィンドウとして開きます

設定パネルはタブに分かれています。変更は **Save** をクリックした後に反映されます。すべての設定は `~/.homunculus/preferences.db` に保存されます。

#### Basic

| 設定 | 説明 | 範囲/タイプ |
|------|------|-------------|
| Name | キャラクターの表示名 | - |
| Scale | キャラクターの表示サイズ | 0.10 - 3.00 |

#### Persona

| 設定 | 説明 |
|------|------|
| Profile | キャラクターの背景とプロフィールの説明（自由テキスト） |
| Personality | 自然言語で記述する性格特性（自由テキスト） |

#### OCEAN

キャラクターのビッグファイブ（Big Five）パーソナリティの各次元を調整します。各特性はスライダーで設定し、レーダーチャートで視覚化されます。

| 特性 | 低い | 高い |
|------|------|------|
| 開放性（Openness） | 保守的 | 好奇心旺盛 |
| 誠実性（Conscientiousness） | 自発的 | 計画的 |
| 外向性（Extraversion） | 内向的 | 外向的 |
| 協調性（Agreeableness） | 独立的 | 協力的 |
| 神経症傾向（Neuroticism） | 安定的 | 敏感 |

### ペルソナ管理ダッシュボード

システムトレイの **「Persona」** エントリから管理ダッシュボードを開きます。この UI ではペルソナの作成、削除、設定が行えます。

## 備考

- ペルソナ MOD は[コンテキストメニュー](./menu)に「Persona」エントリを、システムトレイに「Persona」エントリを追加します。
- ペルソナ MOD は VRMA アニメーションのために [@hmcs/assets](./assets) MOD を必要とします。
- 設定パネルの UI は共有の `@hmcs/ui` コンポーネント（component）ライブラリを使用しています。
