# @hmcs/menu

コンテキストメニュー MOD（`@hmcs/menu`）は、キャラクターを右クリックしたときに表示されるメニューを提供します。利用可能なアクションを表示する WebView ベースの HUD オーバーレイです。

## 概要

**キャラクターを右クリック**するとコンテキストメニューが開きます。メニューにはキャラクター名と利用可能なアクションの一覧が表示されます。アクションは他のインストール済み MOD によって提供されます。例えば、[ペルソナ](./character-settings) MOD は「Persona」エントリを追加します。

**Escape** キーを押すか、メニューの外側をクリックすると閉じます。

## 機能

MOD は `package.json` の `"homunculus"` フィールドにメニューエントリを登録します。各エントリにはラベルと、選択時に実行される MOD コマンドが指定されます。コンテキストメニュー MOD は登録されたすべてのエントリを収集して表示します。

## 備考

- メニューの内容はインストールされている MOD に依存します。新しい MOD をインストールすると、新しいメニューエントリが追加されることがあります。
- メニュー UI は共有の `@hmcs/ui` コンポーネント（component）ライブラリを使用して、統一されたスタイルを実現しています。
