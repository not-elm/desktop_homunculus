# @hmcs/app-exit

App Exit MOD（`@hmcs/app-exit`）は、システムトレイメニューに **Exit** エントリを追加し、Desktop Homunculus をクリーンにシャットダウンできるようにします。

## 概要

アプリケーションを終了するには：

1. OS のメニューバー/システムトレイにある Desktop Homunculus のトレイアイコンをクリック
2. **「Exit」** を選択
3. アプリケーションが正常にシャットダウンされます

内部的には、エンジンの `AppApi::exit()` エンドポイントを呼び出してクリーンなシャットダウンを実行します。

## 備考

- **Windows** では、透明オーバーレイウィンドウに標準的なウィンドウの閉じるボタンがないため、この MOD は特に重要です。トレイの終了メニューがアプリケーションを終了する主な方法となります。
- **macOS** では `Cmd+Q` や Dock メニューからも終了できますが、トレイのエントリはクロスプラットフォームで一貫した終了方法を提供します。
