# shadowPanel

The `shadowPanel` module controls the shadow overlay panel -- a fullscreen transparent layer used for atmospheric effects or focus dimming.

## Import

```typescript
import { shadowPanel } from "@hmcs/sdk";
```

## Functions

| Function | Description |
|----------|-------------|
| [alpha](./alpha) | Gets the current transparency level of the shadow panel |
| [setAlpha](./setAlpha) | Sets the transparency level of the shadow panel |
